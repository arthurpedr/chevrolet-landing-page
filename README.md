## 📦 Como Instalar e Executar

1. Clone o repositório:
```
git clone https://github.com/arthurpedr/Projeto-Yu-Gi-Oh.git
```
2. Acesse a pasta do projeto:
```
cd Projetos-Yu-Gi-Oh
```
3. Abra o arquivo `index.html` no navegador.
